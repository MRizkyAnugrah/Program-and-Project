Nomor 1¶

  1. Secara teori persamaan diferensial manakah yang sesuai untuk memodelkan kasus tersebut? Jelaskan alasan anda.

Teori persamaan differennsial yang sesuai unntuk memodelkan kasus tersebut ialah persamaan eliptik. Persamaan eliptik biasanya digunakan untuk mengkarakterisasi sistem setimbang. Seperti di persamaan Laplace, hal ini ditunjukkan dengan tidak adanya turunan waktu. Dengan demikian, persamaan ini biasanya digunakan untuk menentukan distribusi keadaan tunak dari yang tidak diketahui dalam dua dimensi spasial.

Nomor 3¶

  1. Tuliskan syarat-syarat batas sesuai dengan gambar di atas (atas, bawah dan bagian terisolasi).
  • Titik Terisolasi Atas dan Bawah
$$\frac{\partial^2T}{\partial y^2} = 0$$
  • Titik Terisolasi Kanan
$$\frac{\partial^2T}{\partial x^2} = 0$$
  • Titik Bagian Kiri
$$\frac{\partial^2T}{\partial x^2} + \frac{\partial^2T}{\partial y^2} = 0$$$$T = 120^{\circ}C - 0^{\circ}C = 120^{\circ}C$$
  • Titik Bagian Kanan
$$\frac{\partial^2T}{\partial x^2} + \frac{\partial^2T}{\partial y^2} = 0$$$$T = 0^{\circ}C$$

Nomor 4¶

  1. Tuliskan persamaan numerik untuk menghitung distribusi temperatur pada setiap pada plat (ada 36 titik) dengan menggunakan Metode Beda Hingga;

  • Persamaan Laplace $$\frac{\partial^2T}{\partial x^2} + \frac{\partial^2T}{\partial y^2} = 0$$

  • Central Differences

$$\frac{\partial^2 T}{\partial x^2}=\frac{T_{i+1, j}-2 T_{i, j}+T_{i-1, j}}{\Delta x^2}$$$$\frac{\partial^2 T}{\partial y^2}=\frac{T_{i, j+1}-2 T_{i, j}+T_{i, j-1}}{\Delta y^2}$$
  • Subtitusikan nilai differensial orde 2 ke dalam rumus laplace
$$\frac{T_{i+1, j}-2 T_{i, j}+T_{i-1, j}}{\Delta x^2} + \frac{T_{i, j+1}-2 T_{i, j}+T_{i, j-1}}{\Delta y^2} = 0$$
  • Pada kasus ini nilai $\Delta x = \Delta y$ maka
$$T_{i+1, j}+T_{i-1, j}+T_{i, j+1}+T_{i, j-1}-4 T_{i, j}=0$$$$T_{i, j}=\frac{T_{i+1, j}+T_{i-1, j}+T_{i, j+1}+T_{i, j-1}}{4}$$
  • Misalkan ialah titik i = 0, j = 0, maka:
$$T_{0,0}=\frac{T_{1,0}+T_{-1,0}+T_{0,1}+T_{0,-1}}{4}$$$$T_{0,0}=\frac{T_{1,0}+T_{0,1}}{2}$$

def central_differences(tabel,tabel_lama,i,j,Length,Width,Employ_overrelaxation=1.5):
    a = i + 1
    b = i - 1
    c = j + 1
    d = j - 1
    if a >= Width:
        a -= 2
    if b < 0:
        b += 2
    if c >= Length:
        c -= 2
    if d < 0 :
        d += 2
    list_kosong = np.zeros(1)
    list_kosong = round(1/4*(tabel[a][j] + tabel[b][j] + tabel[i][c] + tabel[i][d]),3)
    hasil = round(Employ_overrelaxation*list_kosong + (1-Employ_overrelaxation)*tabel_lama[i][j],3) 
    return round(hasil,4)

def Error(tabel,tabel_lama,y,x):
    list_kosong = np.zeros(1)
    list_kosong = np.abs((tabel[y][x] - tabel_lama[y][x]) / tabel_lama[y][x])*100
    return list_kosong
